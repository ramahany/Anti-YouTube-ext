### ANTI YOUTUBE EXT
a chrome extension than helps with youtube adection 
## what is does 
=> while watching youtube video no more suggestions are shown in the page, so you dont get distract and can focus on the video you are watching 
=> when opening the home you only see the search bar(no feed), so you can only search for what you need 
=> a on and off button is avalable in the extension so you can enable or disable the service
